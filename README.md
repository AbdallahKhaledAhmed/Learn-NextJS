# Learn-NextJS

This repository is intended to serve as a documentation for me learning Next.js. It contains various examples and tutorials to help me document my learning experience with the framework.

## Getting Started To use this repository, simply clone the branch you are interested in to your local machine and run `npm install` to install the dependencies. You can then start the development server by running `npm run dev`.

## Examples The repository includes a variety of examples demonstrating different aspects of Next.js, including:

- Basic routing
- Static site generation
- API routes
- Serverless functions

Each example is located in its own branch within the `github branches`.
